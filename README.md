## How to View

First, clone the repo and `cd` into it

    git clone https://github.com/kathawala/software-exercise.git
    cd software-exercise

To view in Chrome/Chromium

    chromium --disable-web-security --user-data-dir=/tmp/ index.html
    
To view in Firefox

    firefox index.html
